Basics

Definitions and Representation

Definition 1. (Vertor)

Quantities have both magnitude (the “how much" or “how big" part) and direction are called vectors. Magnitude of a vector is also called length.

Two vectors are equal if they share the same direction and magnitude.

we denote a vector by: \(\vec{a}\) or \(\vec{AB}\) (\(A, B\) are starting point and end point of the vector.)

For maginitude of vector, we denote it as \(|\vec{a}|\), \(|\vec{AB}|\).

Basic Operations

We first introduce 3 (actually, \(2\)) basic operations we can perform on vectors-Adding, Subtruction and multiplication by a scalar.

For adding we have:

As you can see, there are two ways of doing it-head-to-tail or parallelogram method.

For subtruction, we first introducing negetive vector, for \(\vec{a}\), \(-\vec{a}\) is the vector which has the same magnitude but opposite direction with \(\vec{a}\). Hence: \[\vec{a} - \vec{b} = \vec{a} + (-\vec{b})\]

For multiplication with a scalar (or a number), we define:

Definition 2. (Scalar Multiplication) \[\begin{equation*} k\vec{a} = \begin{cases} \text{a vector has same direction as }\vec{a} \text{and the magnitude is } k\times |\vec{a}|, & \text{if } k>0,\\ \text{a vector has opposite direction as }\vec{a} \text{and the magnitude is } -k\times |\vec{a}|, & \text{if } k<0,\\ \vec{0}, & \text{if } k = 0. \end{cases} \end{equation*}\]

By multiplication with a number, there is an interesting and important thing: we call a vector unit vector if the magnitude of it is 1. How do we find a unit vector, denoted by \(\vec{e}_{a}\) or \(\hat{e}_{a}\), which has the same direction as \(\vec{a}\): \[\begin{equation*} \vec{e}_a = \frac{1}{|\vec{a}|} \vec{a} = \frac{\vec{a}}{|\vec{a}|} \end{equation*}\]

Introducing Coordinate System

The great power of vector will unleash only if we equib them with coordinate system, that is analytic geometry. To do so we have to first introduce a concept-basis. We ignore the formal definition of basis, instead, we introduce one set of them:

Definition 3. (\(\vec{i}, \vec{j}, \vec{k}\))

  • \(\vec{i}\)-unit vector with the direction pointing to the positive \(x\)-axis,

  • \(\vec{j}\)-unit vector with the direction pointing to the positive \(y\)-axis,

  • \(\vec{k}\)-unit vector with the direction pointing to the positive \(z\)-axis

The three unit vectors form a basis of 3-D space (in 2-D plane, \(\vec{i}, \vec{j}\) form a basis). Any vectors in the space can be represented by the basis, and the way to represent is unique. We use a graph to illustrate this:

More formally we have: \[\begin{gather*} \vec{a} = a_1 \vec{i} + a_2 \vec{j} + a_3 \vec{k} \quad \text{(in 3-D space)}\\ \vec{a} = a_1 \vec{i} + a_2 \vec{j} \quad \text{(in 2-D plane)} \end{gather*}\] \(a_i\)’s here are called the components of the vector \(\vec{a}\). We seldom write vector in this form, usually we write it as: \[\begin{equation*} \vec{a} = \begin{pmatrix} a_1\\ a_2\\ a_3 \end{pmatrix} \text{ or } \vec{a} = \begin{pmatrix} a_1\\ a_2 \end{pmatrix} \end{equation*}\]

The convenience basis gives us are, first we can calculate the magnitude of a vector: \[\begin{equation*} |\vec{a}| = \sqrt{{a_1}^2 + {a_2}^2 + {a_3}^3} \end{equation*}\]

Second, operations become a simple thing, let \(\vec{a} = \begin{pmatrix} a_1\\ a_2\\ a_3 \end{pmatrix}, \vec{b} = \begin{pmatrix} b_1\\ b_2\\ b_3 \end{pmatrix}\): \[\begin{equation*} \vec{a} \pm \vec{b} = \begin{pmatrix} a_1 \pm b_1\\ a_2 \pm b_2\\ a_3 \pm b_3 \end{pmatrix},\quad k \vec{a} = \begin{pmatrix} ka_1\\ ka_2\\ ka_3 \end{pmatrix} \end{equation*}\]

And now for \(\vec{e}_{a}\), we have: \[\begin{equation*} \vec{e}_{a} = \frac{\vec{a}}{|a|} = \frac{1}{\sqrt{{a_1}^2 + {a_2}^2 + {a_3}^2}} \begin{pmatrix} a_1\\ a_2\\ a_3 \end{pmatrix} = \begin{pmatrix} \frac{a_1}{\sqrt{{a_1}^2 + {a_2}^2 + {a_3}^2}}\\ \frac{a_2}{\sqrt{{a_1}^2 + {a_2}^2 + {a_3}^2}}\\ \frac{a_3}{\sqrt{{a_1}^2 + {a_2}^2 + {a_3}^2}} \end{pmatrix} \end{equation*}\]

Finally, we introduce two terms:

  • positional vector: the vector from the origin to the point

  • displacement vector of point B relative to A: vector from point A to point B

A thing that we use frequently: Let vector \(\vec{a}\) denote the positional vector of point A, and \(\vec{b}\) for that of point B, then we have: \[\begin{equation*} \vec{AB} = \vec{b} - \vec{a} \end{equation*}\]

Parallelism

The things we need to know is extremely easy, just:

Theorem 1. (Parallel)

\[\begin{equation*} \vec{a} \parallel \vec{b} \Rightarrow \vec{b} = k\vec{a}. \end{equation*}\]

Just don’t forget it (because there are so many formulas of vectors.)

And we can use this to identify if three points are colinear or not:

Theorem 2. (colinear)

Points \(A, B\) and \(C\) are colinear if \(\vec{AB} = k\vec{BC}\).

Scalar Product (Inner Product or Dot Product)

We define the so-called scalar product (or inner product, dot product) between two vectors as:

Definition 4. (Scalar Product, Inner Product, Dot Product) \[\begin{equation*} \vec{a}\cdot \vec{b} = |\vec{a}||\vec{b}|\cos(\theta) \end{equation*}\] here, \(\theta\) is the angle between vector \(\vec{a}\) and \(\vec{b}\). Actually, if you are careful, you will find that there are two angles if you put two vectors’ tails together. We always chose the angle which is in \([0, \pi]\).

Finally, unlike for numbers, the dot here can not be ignore.

The are some pictures we can refer to:

Once we introduce scalar product, we could have these two formulas: \[\begin{equation*} \vec{a}\cdot \vec{a} = |\vec{a}|^2 \Leftrightarrow |\vec{a}| = \sqrt{\vec{a}\cdot \vec{a}},\qquad \cos(\theta) = \frac{\vec{a}\cdot \vec{b}}{|\vec{a}||\vec{b}|} \end{equation*}\] Thus, we could find the length of a vector and angle between two vectors.

A directly corollary of the second formula is:

Theorem 3. (Perpendicular) \[\begin{equation*} \vec{a} \perp \vec{b} \Leftrightarrow \vec{a} \cdot \vec{b} = 0 \end{equation*}\]

Finally, if we know the components of vectors in terms of \(\{\vec{i}, \vec{j}, \vec{k}\}\), we can compute scalar product in such a way:

Theorem 4. Suppose we have two vectors: \[\vec{a} = \begin{pmatrix} a_1\\ a_2\\ a_3 \end{pmatrix}, \quad \vec{b} = \begin{pmatrix} b_1\\ b_2\\ b_3 \end{pmatrix}\] Then, we have: \[\begin{equation*} \vec{a}\cdot \vec{b} = a_1b_1 + a_2b_2 + a_3b_3 \end{equation*}\]
