Exponential Functions

definitions

Definition 1. (Exponential Functions)

We call functions like \(y = a b^x (b >0)\) exponential functions. Its domain is all the real numbers, and its range is \((0, +\infty) (y>0)\).

Specifically

  • if \(a>0, 0<b<1\), we call it exponential decay functions, and say \(b\) is the decay factor;

  • if \(a>0, b>1\), we call it exponential growth functions, and say \(b\) is the growth factor

Consider two simple exponential functions: \(y = 2^x, y = \left( \frac{1}{2}\right) ^x\):

As we can see:

  • \(y=2^x\) rises from left to right, while \(y = \left( \frac{1}{2}\right)^x\) falls from left to right,

  • Both of them pass the point \((0, 1)\),

  • \(y=2^x\) pass \((1, 2)\), and \(y = \left( \frac{1}{2}\right)^x\) pass \(1, \frac{1}{2}\) (Generally, \((1, b)\)).

Definition 2. (Asymptote)

An asymptote is a line which that a graph approaches more and more closely.

According to the defintion of asympotote and the graph of \(y = b^x\), we find that:

Theorem 1. The asymptote of \(y = b^x\) is \(y = 0\).

Applications

Definition 3. (Exponential Growth Model)

\[\begin{equation*} y = a (1 + r)^t \end{equation*}\]

Examples of exponential growth in real life include: bank account balance, Spread of Infectious Diseases (Early Stage) etc.

Definition 4. (Exponential Decay Model)

\[\begin{equation*} y = a (1 - r)^t \end{equation*}\]

Examples of exponential decay in real life include: Radioactive Decay, depreciation etc.

Definition 5. (Compound Interest)

\[\begin{equation*} A = P \left( 1 + \frac{r}{n}\right) ^{nt} \end{equation*}\]

Here, \(P\)-principal (initial amount of money), \(r\)-annual rate, \(n\)-times compounded in a year (or other period), \(t\)-time, usually year.

The Number \(e\)

First thing first, \(e\) is a number, its value is approximately \(2.71828\dots\) (just like \(\pi\) just represents a number which is \(3.1415926\dots\)).

Consider a sequence \(\{\left( 1 + \frac{1}{n} \right) ^ n\}\): \[\left( 1 + \frac{1}{1} \right) ^ 1, \left( 1 + \frac{1}{2} \right) ^ 2, \left( 1 + \frac{1}{3} \right) ^ 3, \cdots, \left( 1 + \frac{1}{n} \right) ^ n,\cdots\] If we go to evaluate the value of each item in this sequence, we will find that as \(n\) gets bigger, the value of the items will get more and more closer to the value of number \(e\).

Definition 6. (continuously componded interest)

We’ve seen formula for componded interest: \[\begin{equation*} A = P \left( 1 + \frac{r}{n}\right) ^{nt} \end{equation*}\] What if \(n\) get infinitely large? It turns out, it will be: \[\begin{equation*} A = Pe^{rt} \end{equation*}\]

Logarithmic Functions

definition

We know that \(2^5 = 32\), what if we were facing such a question: what is the index, if the base number is 2, and the outcome is 32? The answer is \(5\), of course, but based on this we have some new ideas:

Definition 7. (Logrithm of Base B)

\[\begin{equation*} b^a = c \Leftrightarrow \log_{b}(c) = a \end{equation*}\]

So, when we are evaluating things like: \(\log_{3}(27)\), we are asking such a question: \(3\) to what power gives 27.

Some common notations (and you can see them in your calculator): \[\begin{equation*} \log(c) = \log_{10}(c)\qquad \ln(c) = \log_{3}(c) \end{equation*}\]

Properties

There are some common properties about logrithm: \[\begin{align*} &\log_{b}(1) = 0 &b^0 = 1\\ &\log_{b}(b) = 1 &b^1 = b\\ &b^{\log_{b}(c)} = c &\text{Think about the definition of logrithm}\\ &\log_{b}(mn) = \log_{b}(m) + \log_{b}(n) &b^j = m, b^k = n, b^j \cdot b^k = b^{j+k} = mn\\ &\log_{b}(\frac{m}{n}) = \log_{b}(m) - \log_{b}(n) &b^j = m, b^k = n, \frac{b^j}{b^k} = b^{j-k} = \frac{m}{n}\\ &\log_{b}(m^n) = n\log_{b}(m) & b^j = m, (b^j) ^n = b^{nj} = b^{n\log_{b}(m)} = m^n \end{align*}\]

And there are one especially important property:

Theorem 2. (Change of Base Formula)

\[\begin{equation*} \log_{a}(c) = \frac{\log_{b}(c)}{\log_{b}(a)} \end{equation*}\]

Logarithmic Functions

Definition 8. (Logarithmic Function) We call functions like: \[\begin{equation*} y = \log_{b}(x) \quad(b>0) \end{equation*}\] The domain of it is \(x > 0\), and the range of it is all the real numbers.

If you give a number, say \(j\) to be the index you can get a number: \(b^j = k\), this is how we use the exponential function. Reversely, if you give the number which is the outcome \(k\) to the logarithm of base \(b\), it give you the index i.e. \(\log_{b}(k) = j\). So, The logarithmic function is the inverse function of the exponential function with the same base number. We can also verify this by the definition of inverse functions.
